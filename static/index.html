Glitchi-Stream v2.0 — Mobile, Speed & Polish

  • Mobile UI redesigned — Bottom tab bar with glowing active indicator, floating mini-player with accent-colored play button, touch-optimized cards with :active scale feedback, 2-column search grid, larger tap targets (44px+) throughout, safe-area insets for notched phones

  • Playlist download now auto-zips and delivers to device — batch playlist downloads collect all completed filenames into a ZIP and trigger browser download automatically
  • Web Worker polling — download progress runs in a dedicated thread, never throttled when browser tab is hidden; all setInterval/setTimeout replaced with worker-driven events
  • Playlist bugs fixed — openPlaylistView classList mismatch (close button now works), smarter local file matching (artist + track name word overlap), clear toast when tracks aren't downloaded yet
  • Search speed doubled — debounce reduced from 400ms to 250ms, client-side result cache with 60s TTL and 50-entry limit for instant repeat searches

  • 8-second timeout on SpotAPI calls — page still loads if SpotAPI is slow or unreachable
  • 18-theme system preserved across all pages with smooth CSS custom property transitions
  • Spotify track/album/artist detail preview with download + add-to-playlist actions
  • All dead CSS purged — removed browse-specific rules after Home tab migration

📝 DEV Notes

Current Architecture

Backend: FastAPI with SpotAPI for Spotify metadata — no API keys required

Downloads: yt-dlp / SpotDL via asyncio subprocess, task-based progress tracking with semaphore (max 3 concurrent), always MP3 320kbps

Playback: Local file playback via browser Audio element

Frontend: Vanilla JS, no framework, no build step — static files served directly

Themes: 18 CSS custom property themes with data-theme attribute switching

Worker: Web Worker polls /download/progress every 1s in a separate thread

Key Files

app/main.py — FastAPI entry point, routers, static mount

app/api/ — Download, search, playlists, browse endpoints

app/services/ — SpotiFLAC/yt-dlp/SpotDL download logic, search, task manager

static/app.js — Entire frontend (routing, search, downloads, playlists, themes)

static/styles.css — All 18 theme definitions + responsive layout

static/index.html — SPA shell (sidebar, tabs, modals, mini-player)

static/download-poll-worker.js — Web Worker for background-safe download progress
